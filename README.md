# Movie App

I create this app for learn develop application using React, Typescript, Vite, React Router, and Redux Toolkit

# API Service

[Movie Database API](https://api.themoviedb.org)

## Installation & Usage

```javascript
# Installation
npm install (node version 18)

# running development
npm run dev

# format code
npm run format

# linting code
npm run lint

# build application
npm run build

# run builds
npm run preview
```
